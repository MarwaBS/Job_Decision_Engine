# Job Decision Engine — pinned runtime dependencies.
#
# Pins are exact (==) per the lesson from the sibling nyc-real-estate-predictor
# project: loading pickled / trained artefacts under a different library
# version can silently corrupt outputs. We ship the exact wheel that the
# system was built + tested against.
#
# How to regenerate:
#   1. make a clean venv: python -m venv .venv && .venv\Scripts\activate
#   2. pip install -r requirements.txt
#   3. python -m pytest tests/            (must pass)
#   4. bump a version only when all tests are green.
